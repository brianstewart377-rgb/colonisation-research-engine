# Project roadmap

This roadmap treats Colonisation Research Engine as a long-term research platform.

## Phase 1

Knowledge

Establish the constitution, ontology, source catalog, architecture, planner safety rules, and research versioning model.

## Phase 2

Mechanics Catalogue

Start a structured mechanic-by-mechanic catalogue with confidence, evidence, contradictions, and patch history.

## Phase 3

Experiments

Define and record repeatable experiment workflows, including failed predictions and null results.

## Phase 4

Database

Design the database schema and provenance model needed to store sources, observations, mechanics, experiments, contradictions, and planner-safe knowledge projections.

## Phase 5

Evidence Ingestion

Add controlled ingestion for official mechanics sources, bulk galaxy/state sources such as Spansh and EDSM, and curated manual evidence entry.

## Phase 6

Research Dashboard

Build a review surface for mechanics confidence, contradiction queues, evidence trails, and experiment outcomes.

## Phase 7

AI Planner

Build a planner that consumes verified knowledge only and emits confidence-labeled plans with caveats and suggested experiments.

## Phase 8

Community Evidence

Introduce community-facing contribution processes, likely using GitHub Discussions for research debate and evidence challenge while keeping Issues for work tracking.

## Phase 9

Model Fine Tuning

Consider fine-tuning only after the repository contains a strong evidence graph, verified mechanics catalogue, contradiction history, and enough structured experiments to justify it.
