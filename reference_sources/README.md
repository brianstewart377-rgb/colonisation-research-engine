# Reference Sources

Canonical reference documents used for Colonisation Research Engine knowledge extraction.

These are distinct from live evidence:

- `reference_sources/` contains community/reference/source documents used to extract claims.
- `evidence/` contains live screenshots, observations, experiments, and evidence records.

Every extracted claim should cite a stable source ID such as `MG-0001`, `FW-0001`, `DM-0001`, or `DD-0001`, plus page/sheet/section where possible.
