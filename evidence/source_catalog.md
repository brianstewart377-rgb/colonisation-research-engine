# Source catalog

This document summarizes the most important evidence source families currently identified in `colonisation_ai_data_sources_review.md`.

## Official mechanics sources

- Frontier official System Colonisation Guide
- Frontier Trailblazers launch and balance patch notes
- Frontier Dodec and later colonisation patch notes
- Official journal manual

These are the strongest sources for intended mechanics, patch drift, and official terminology.

## Journal and schema sources

- Official journal-side files
- Jixxed journal schemas
- EDDN journal schema documentation
- EDMarketConnector ecosystem docs

These define what can actually be observed from player-side evidence and ecosystem exports.

## Bulk galaxy and station sources

- Spansh dumps
- EDSM nightly dumps
- EDDN live stream
- ED Galaxy Data historical archives

These are the strongest candidates for current-state and historical galaxy context.

## Public validation sources

- Inara public star system pages
- Inara station pages
- Inara architect pages
- Inara public logbooks
- Raven Colonial public system pages

These are valuable for named examples, visible build states, planners, and public corroboration.

## Community interpretation sources

- CMDR Mechan’s Colonization Mega Guide
- DaftMav planning spreadsheet
- SrvSurvey wiki
- BGS-Tally colonisation tracking
- Raven Colonial release thread and related tooling

These are useful for heuristics, hypotheses, and workflow understanding, but they are not canonical truth.

## Source handling rules

- Raw source data must remain separate from parsed observations.
- Public page evidence is useful even when API access is unavailable.
- Narrative evidence should be preserved but confidence-labeled.
- Same-name facilities across systems must be treated as possible collisions rather than automatic matches.
